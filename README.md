# tdp2-tp0-backend

## Levantar la API de forma local
```sh
$ git clone https://github.com/christiangelone/tdp2-tp0-backend.git
$ cd tdp2-tp0-backend
$ npm install
$ npm start
```
La API se va a levantar en el puerto 3333 (http://localhost:3333/)
